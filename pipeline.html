OpenGL Pipeline Map

OpenGL Insights includes a detailed OpenGL pipeline map as a two-sided 14x18 inch detachable color poster. The map cross-references the OpenGL 4.2 and ES 2.0 specs. The ES pipeline is relevant to both ES and WebGL developers.

For the benefit of the entire community, the pipeline map included with OpenGL Insights is available below along with the pipeline map for more recent versions of OpenGL and OpenGL ES.
